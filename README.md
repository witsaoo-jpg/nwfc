# Nursing Workforce Calculation System (NWFC)

ระบบคำนวณอัตรากำลังพยาบาลตามภาระงาน
พัฒนาเพื่อใช้ในการบริหารอัตรากำลังทางการพยาบาล

## วัตถุประสงค์
- คำนวณอัตรากำลังพยาบาลตามภาระงานจริง
- สนับสนุนการตัดสินใจเชิงบริหาร

## กลุ่มผู้ใช้งาน
- หัวหน้าหอผู้ป่วย
- ผู้บริหารทางการพยาบาล
- นักวิจัยด้านการพยาบาล

## เทคโนโลยี
- HTML / CSS / JavaScript
- Progressive Web App (PWA)
- GitHub Pages

## สถานะลิขสิทธิ์
โปรแกรมนี้เป็นผลงานที่พัฒนาขึ้นโดยผู้วิจัย
จดลิขสิทธิ์แล้ว ไม่อนุญาตให้นำไปใช้เชิงพาณิชย์โดยไม่ได้รับอนุญาต# nwfc
Nursing Workforce Calculation System
